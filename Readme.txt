1.open menu.py and run
2.from apps icon on top left corner you could choose:
	i.Truth table: input number of laterals in minterm, click on enter truth table size. Then enter the minterm numbers you want to include in your function, and click enter minterms
	ii.Simplifier: enter minterms/laterals you want to simplify together, then click calc
	iii.read: enter a truth table containing inputs and outputs, in which its minterms would be extracted and simplified
	iv.equation: enter a boolean equation and truth table would be output
